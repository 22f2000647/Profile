# Profile

## Run locally

Use the helper script:

```bash
./serve.sh
```

Or specify a custom port:

```bash
./serve.sh 8080
```

Then open:

- `http://127.0.0.1:8000/index.html` (or your chosen port)

> Note: If you're inside a container/VM, `127.0.0.1` refers to that environment, not your host machine. Use the forwarded port URL from your IDE/container runtime.
